## Desenvolvimento

1. Instale as dependências:

   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```

3. Para gerar uma versão de produção:

   ```bash
   npm run build
   ```

O conteúdo deste site está disponível em português.
